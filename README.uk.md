# Kapot Tracker

[English](README.md) · **Українська**

Бортовий журнал і облік витрат для водіїв — паливо, ТО, ремонти й документи в одному
місці, з розпізнаванням чеків з фото, розумними нагадуваннями та Telegram-ботом.
Mobile-first PWA, українською та англійською.

**Живий застосунок:** https://kapot-tracker.vercel.app · **API:** https://kapot-tracker.duckdns.org

---

## Що вміє

### Журнал і записи
- Кілька авто на акаунт; чотири типи записів — **заправка, ТО, ремонт, витрата**.
- **Розхід пального «повний-до-повного»** (л/100 км) з історії заправок.
- Фото до кожного запису (чеки, наряди) — галерея + лайтбокс, до 10 МБ.
- Редагування, **дублювання** («повторити останню»), видалення, повнотекстовий пошук
  разом із фільтром за типом.
- **Офлайн-запис**: зберігається локально й синхронізується, коли зʼявляється звʼязок.

### OCR з фото та AI (Google Gemini)
- Скан **чека АЗС**, **наряду СТО** або **будь-якого чека витрати** — фото автозаповнює
  форму. Vision-first (Gemini читає криві фото з телефона будь-якою мовою), а безкоштовні
  tesseract / OCR.space — як фолбек без ключа.
- Мʼяка деградація: коли модель перевантажена, застосунок каже **«скан недоступний,
  введіть вручну»**, а не звинувачує фото.
- **Вільний текст у Telegram-боті** — «залив 40 л на окко за 2200»: модель сама
  визначає дію й заповнює правильний запис.

### Аналітика та інсайти
- Витрати **за категоріями й місяцями**, **прогноз** витрат, вартість володіння (**₴/км**).
- Динаміка розходу, **історія цін** на паливо, статистика по АЗС і **найдешевша** заправка.
- **Місячний бюджет** із темпом і прогнозом; вартдог **стрибка розходу**
  («+18 % над нормою — перевірте тиск у шинах»).
- **«Ваш рік з Kapot»** — річний підсумок у стилі Spotify Wrapped із карткою для шерингу.
- **PDF-звіт сервісної історії** та **CSV**-експорт журналу.

### Нагадування та центр сповіщень
- **Сервісні інтервали** (за пробігом та/або часом) з прогнозом дати й кнопкою «Виконано»
  в один тап, що записує ТО й зсуває відлік уперед.
- **Шини**: сезонна заміна (регіон із номера), ротація вісей кожні ~10 000 км і
  **вік шин** («їм уже 9 років — перевірте або замініть»).
- Нагадування про **строк документів / ОСЦПВ** з ескалацією в останній тиждень.
- **Центр сповіщень у застосунку** (нічого не зберігається — рахується на льоту), банер на
  дашборді, **бейдж на іконці** й **Telegram-пуш** із тижневим підсумком.

### Записи про авто
- **Документи** з відстеженням строків (страховка, техогляд…).
- Публічний **QR-паспорт** авто (сервіс/паркування) і модуль **шин**.
- **OBD**-діагностика з CSV Car Scanner і **тех. довідка** (моменти затяжки, обʼєми) по авто.

### Спільний доступ, дані, акаунти
- **Спільне ведення** авто (власник / редактор / глядач) з інвайт-лінками та підписом
  автора на кожному записі.
- Повний **бекап і відновлення JSON** (авто, конфіг, специфікації, шини, історія) та CSV.
- JWT (access + refresh, відкликання через token-version), **скидання пароля через
  Telegram**, rate limiting і повне **видалення акаунта**.
- Alembic-міграції застосовуються автоматично на старті.

### Платформа
- **PWA** — встановлюється, працює офлайн, бейдж на іконці.
- **Дві мови** — українська та англійська.
- **Telegram-бот** — швидкі записи, нагадування, тижневий підсумок, бекап на вимогу.

## Архітектура

```
frontend (React PWA, Vercel / nginx)  ──/api──►  backend (FastAPI)  ──►  PostgreSQL / SQLite
                                                       ▲
                                        Telegram-бот (aiogram, той самий образ)
```

- **Backend** — FastAPI (Python 3.12), REST + JWT. SQLite для локальної розробки,
  PostgreSQL у Docker/проді (SQLAlchemy). Alembic-міграції на старті.
- **Frontend** — React + Vite + Tailwind як встановлюваний PWA (vite-plugin-pwa).
- **Бот** — aiogram v3, окремий процес із того самого backend-образу.
- **OCR** — Google Gemini vision (ключ опційно), tesseract + OCR.space як безкоштовний фолбек.
- **Хостинг** — фронт на Vercel; backend на Oracle Cloud Always-Free VM за Caddy, DNS через
  DuckDNS. На невеликому масштабі працює безкоштовно.

### Структура репозиторію

```
backend/
  app/
    routers/     # HTTP-ендпойнти (auth, cars, logs, analytics, tires, notifications, ocr, …)
    services/    # доменна логіка (stats, forecast, fuel, ocr, climate, notifications, year_review, …)
    bot/         # aiogram-хендлери, нагадування, AI-розбір вільного тексту
    models.py    # моделі SQLAlchemy
    schemas.py   # схеми Pydantic
  alembic/       # міграції БД
frontend/
  src/
    views/       # сторінки (Dashboard, Logbook, Analytics, Tires, YearReview, Notifications, …)
    components/  # спільний UI
    api/         # тонкий API-клієнт на ресурс
    i18n/        # локалі uk / en
docs/            # DEPLOY.md і дизайн-нотатки
```

## Як запустити

Потрібні Docker + Docker Compose.

```bash
cp .env.example .env                     # відредагуй — щонайменше SECRET_KEY
docker compose up -d --build             # db + backend + frontend
docker compose --profile bot up -d       # (опційно) Telegram-бот
```

- Frontend: http://localhost (nginx) · API: http://localhost:8000
- Локальна розробка без Docker: `uvicorn app.main:app --reload` (backend) і `npm run dev` (frontend).

### Ключова конфігурація (`.env`)

| Змінна | Призначення |
|---|---|
| `SECRET_KEY` | Підпис JWT — **обовʼязково** сильне випадкове значення |
| `DATABASE_URL` | `sqlite:///./kapot_tracker.db` (dev) або `postgresql+psycopg2://…` |
| `PUBLIC_URL` | Публічний origin фронта (magic-лінки, QR-паспорт, CTA бота) |
| `CORS_ORIGINS` | Дозволені origins через кому |
| `GEMINI_API_KEY` / `GEMINI_MODEL` | Vision-OCR + розбір вільного тексту в боті (опційно) |
| `TELEGRAM_BOT_TOKEN` / `TELEGRAM_BOT_USERNAME` | Telegram-бот (опційно) |
| `BAZA_GAI_API_KEY` | Пошук авто за номером/VIN (Україна, опційно) |
| `SMTP_*` | Підтвердження пошти й скидання пароля (обовʼязково при `ENV=production`) |
| `UPLOADS_DIR` / `BACKUP_DIR` / `BACKUP_KEEP` | Сховище фото й бекапи БД |

Усе опційне деградує мʼяко: нема ключа Gemini → безкоштовні OCR-рівні; нема токена бота →
нема Telegram; нема SMTP у dev → реєстрація автопідтверджується.

Повне налаштування проду (Oracle Cloud + DuckDNS + Caddy, безкоштовно):
**[docs/DEPLOY.md](docs/DEPLOY.md)**.
